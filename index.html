RELACIONES ENTRE ASIGNATURAS

Las asignaturas se centran en el desarrollo de software y sistemas, con la programación como núcleo, ya que se crean aplicaciones que se conectan a bases de datos y funcionan en distintos sistemas operativos. Los lenguajes de marcas son útiles para organizar la información en línea, y los entornos de desarrollo ayudan a crear y probar el código. Además, el inglés técnico es fundamental para comprender la documentación, mientras que FOL brinda habilidades prácticas que serán esenciales en el ámbito profesional.

Sistemas Informáticos

Se centra en la instalación, configuración y gestión de sistemas y redes. Esto les permite entender cómo sus aplicaciones interactúan con el hardware y las redes, y es clave para administrar servidores que alojan bases de datos y aplicaciones.

  • Explotación de sistemas microinformáticos
  • Instalación de sistemas operativos
  • Gestión de la información
  • Conexión de sistemas en red
  • Gestión de recursos en una red
  • Explotación de aplicaciones informáticas de propósito general

Bases de Datos

Las bases de datos son fundamentales para organizar, almacenar y recuperar información. Están ligadas a la programación, ya que las aplicaciones deben manejar datos de manera eficiente. Comprender su funcionamiento permite diseñar aplicaciones que gestionen información eficazmente.

  • Almacenamiento de la información
  • Bases de datos relacionales
  • Realización de consultas
  • Tratamiento de datos
  • Programación de bases de datos
  • Interpretación de diagramas entidad/relación
  • Uso de bases de datos objeto-relacionales

  • Aplicación de lenguajes de marcas a la sindicación de contenidos

  • Almacenamiento de información

Programación

La Programación es el pilar del desarrollo de software, permitiendo crear aplicaciones dinámicas. Enseña a manipular datos y estructurar contenido, siendo clave para todas las asignaturas, ya que sirve de base para desarrollar aplicaciones y sistemas informáticos.

  • Identificación de los elementos de un programa informático
  • Utilización de objetos
  • Uso de estructuras de control
  • Desarrollo de clases
  • Lectura y escritura de información
  • Aplicación de las estructuras de almacenamiento
  • Uso avanzado de clases
  • Mantenimiento de la persistencia de los objetos
  • Gestión de bases de datos relacionales

Entornos de desarrollo

Los Entornos de Desarrollo son plataformas esenciales que facilitan escribir, probar y depurar código de manera eficiente.

  • Desarrollo de software

Inglés Técnico

El Inglés Técnico es esencial en informática, ya que la mayoría de la documentación y herramientas están en este idioma. Comprenderlo es fundamental para un posible futuro en una empresa internacional.

Historia de EJemplo

Imagineu un autònom que el contracten (FOL) per a programar la part visual de la pàgina web (LLENGUATGE DE MARQUES). Aquesta pàgina será una página privada disponible sols per a la empresa que servirá per a comprar automáticamente els uns materials de obra a un proveedor per a revendrels, aquesta página estarà en anglès (ANGLÈS), contindrà una base de dades (BASES DE DADES) amb tots els empleats, materials y packs de material que un altre programador contractat programarà (ENTORNS DE DESENVOLUPAMENT). En el contracte, té un pressupost de 1200 € per a comprar un ordenador (SISTEMES INFORMÀTICS) en el qual treballa. A part altre programador fará el apartat del backend per a automatitzar-ho tot utilitzant java (PROGRAMACIÓ)Imagina un autónomo al que contratan (FOL) para programar la parte visual de una página web (LENGUAJE DE MARCAS). Esta página será privada y estará disponible únicamente para la empresa, sirviendo para comprar automáticamente materiales de obra a un proveedor con el fin de revenderlos. La página estará en inglés (INGLÉS) y contendrá una base de datos (BASES DE DATOS) con todos los empleados, materiales y paquetes de materiales que programará otro programador contratado (ENTORNOS DE DESARROLLO). En el contrato, el autónomo tiene un presupuesto de 1200 € para comprar un ordenador (SISTEMAS INFORMÁTICOS) con el que trabajará. Además, otro programador se encargará de la parte del backend para automatizar todo utilizando Java (PROGRAMACIÓN).
